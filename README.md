# Tailwind Login Page

A simple and responsive login page built using HTML, Tailwind CSS, and JavaScript.

## Overview

This project provides a clean and minimal login interface with basic client-side form validation. It is designed for learning purposes or for integration into a larger web application.

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)

## Features

- Responsive design
- Minimal and modern layout
- Basic form validation
- Easy to customize and extend
